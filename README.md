gatling-sbt
===========

A sbt template project for writing [Gatling](https://github.com/excilys/gatling.git) simulations in Eclipse.

Depends on :

* gatling 1.1.6

* sbt 0.11.3

Scala 2.9.2

To generate eclipse project:

sbt eclipse

To run basic example:

* sbt run
* Choose engine 
* Data is generated in the results folder
